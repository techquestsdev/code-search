## Frequently Asked Questions

### How do I get started with Code Search?

Follow our [Quick Start guide](/getting-started/quick-start/) to get Code Search running with Docker in a few minutes. You can connect your first code host and start searching right away.

### What code hosts are supported?

Code Search supports **GitHub** (cloud and Enterprise), **GitLab** (cloud and self-hosted), **Gitea**, and **Bitbucket**. You can connect multiple code hosts simultaneously.

### Is there a self-hosted option?

Yes — Code Search is designed to be self-hosted. You run it on your own infrastructure and your code never leaves your network. See the [installation guide](/getting-started/quick-start/) for details.

### How does pricing work?

The **Community** edition is free and open source with no user limits or feature restrictions. Enterprise features (SSO, RBAC, audit logging) are coming soon.

### What's included in the Enterprise plan?

Enterprise features — including SSO/OIDC authentication, role-based access control (RBAC), audit logging, and dedicated support — are coming soon. Contact us using the form above to learn more.

### Where do I report bugs or request features?

Open an issue on our [GitHub repository](https://github.com/techquestsdev/code-search/issues). We actively triage and respond to community issues.

### How do I get technical support?

For Community users, support is available via [GitHub Issues](https://github.com/techquestsdev/code-search/issues). Enterprise customers get priority support — reach out using the form above or email us directly.
