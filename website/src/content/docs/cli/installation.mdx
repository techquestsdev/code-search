## Installation Methods

<Tabs>
  <TabItem label="macOS" icon="apple">
    ### Homebrew (Recommended)

    ```
    bash brew install aanogueira/tap/code-search
    ```

    ### Direct Download

    ```
    bash # Apple Silicon (M1/M2/M3) curl -L
    https://github.com/techquestsdev/code-search/releases/latest/download/code-search-darwin-arm64
    -o code-search # Intel curl -L
    https://github.com/techquestsdev/code-search/releases/latest/download/code-search-darwin-amd64
    -o code-search chmod +x code-search sudo mv code-search /usr/local/bin/
    ```
  </TabItem>

{" "}
  <TabItem label="Linux" icon="linux">
    ### Direct Download

    ```
    bash # x86_64 curl -L
    https://github.com/techquestsdev/code-search/releases/latest/download/code-search-linux-amd64
    -o code-search # ARM64 curl -L
    https://github.com/techquestsdev/code-search/releases/latest/download/code-search-linux-arm64
    -o code-search chmod +x code-search sudo mv code-search /usr/local/bin/
    ```

    ```
    bash curl -sSL https://code-search.techquests.dev/install.sh | bash
    ```

  <TabItem label="Windows" icon="seti:windows">

  ### PowerShell

    ```powershell # Download Invoke-WebRequest -Uri

    "<https://github.com/techquestsdev/code-search/releases/latest/download/code-search-windows-amd64.exe>"
    -OutFile "code-search.exe" # Move to PATH Move-Item code-search.exe
    C:\Windows\System32\
    ```

    ### Scoop

    ```powershell scoop bucket add code-search
    <https://github.com/aanogueira/scoop-bucket> scoop install code-search
    ```

<TabItem label="Docker" icon="seti:docker">

### Run from Docker

  ```bash
  docker run --rm -it

  ghcr.io/techquestsdev/code-search-cli:latest --help
  ```

  ### Create an alias

  ```bash alias code-search='docker run --rm -it -v
  ~/.code-search:/root/.code-search ghcr.io/techquestsdev/code-search-cli:latest'
  ```

</TabItem>

  <TabItem label="Golang" icon="seti:go">
    ### From Source

    ```bash
    go install github.com/techquestsdev/code-search/cmd/cli@latest
    ```

    This installs to

    `$GOPATH/bin/cli`. You may want to rename it:

    ```bash
    mv $GOPATH/bin/cli
    $GOPATH/bin/code-search
    ```
  </TabItem>

## Verify Installation

Check that the CLI is installed correctly:

```bash
code-search --version
# code-search version 1.0.0

code-search --help
```

## Shell Completions

Enable command-line completions for your shell:

<Tabs>
  <TabItem label="Bash">
    ```bash
    # Add to ~/.bashrc
    source <(code-search completion bash)
    ```
  </TabItem>

  <TabItem label="Zsh">
    ```bash
    # Add to ~/.zshrc
    source <(code-search completion zsh)

    # Or generate to fpath
    code-search completion zsh > "${fpath[1]}/_code-search"
    ```
  </TabItem>

  <TabItem label="Fish">
    ```bash
    code-search completion fish | source

    # Or save to completions directory
    code-search completion fish > ~/.config/fish/completions/code-search.fish
    ```
  </TabItem>

  <TabItem label="PowerShell">
    ```powershell
    code-search completion powershell | Out-String | Invoke-Expression

    # Or add to profile
    code-search completion powershell >> $PROFILE
    ```
  </TabItem>
</Tabs>
