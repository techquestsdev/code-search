## Frequently Asked Questions

### What's included in Community?

The Community edition is the full open-source Code Search platform. You get lightning-fast code search powered by Zoekt, bulk find & replace with automatic MR/PR creation, CLI and MCP server for AI assistants, SCIP-powered code intelligence, and support for all major code hosts (GitHub, GitLab, Gitea, Bitbucket). There are no user limits or feature restrictions.

### What do enterprise tiers add?

Enterprise tiers add security and compliance features on top of the Community edition: SSO via any OpenID Connect provider (Okta, Azure AD, Google Workspace, etc.), role-based access control with glob-pattern repository permissions, audit logging of all searches and operations, and cryptographically signed license management.

### How does licensing work?

Enterprise licenses are cryptographically signed with Ed25519. Each license specifies the customer name, enabled features, user limits, and expiration date. The enterprise binary validates the license at startup — no phone-home or external license server required.

### Can I upgrade between tiers?

Yes. You can upgrade from Community to any enterprise tier at any time. Your existing configuration, database, and indexes are fully compatible — the enterprise binary is a drop-in replacement for the core API server.

### How is the user limit counted?

User limits apply to the number of unique authenticated users in the system. Users are counted based on OIDC identity. The Community edition has no user tracking since it doesn't include authentication.
